# chat-app-front

シンプルなチャットアプリのサンプル

## コマンド

```
// ローカルサーバー起動
npm run dev

// ユニットテストの実行
npm run test:unit
```

## 使用技術

- Vue 3
- Vue Router
- Vuex
- TypeScript
- Vite
- Jest
- Tailwind CSS

## 開発環境

- VSCode
- Prettier
- ESLint
- [Volar](https://marketplace.visualstudio.com/items?itemName=johnsoncodehk.volar)
